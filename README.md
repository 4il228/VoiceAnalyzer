# Анализатор вокального диапазона

Клиентское веб-приложение для анализа вокального диапазона по микрофону. Пользователь поёт нижнюю и верхнюю комфортные ноты, приложение определяет высоту тона, фиксирует стабильные ноты и показывает итоговый диапазон, примерную классификацию голоса и похожие известные голоса.

Всё работает локально в браузере. Бэкенд, серверные API и внешние сервисы обработки аудио не используются.

## Возможности

- Захват звука с микрофона через Web Audio API.
- Выбор устройства захвата.
- Определение pitch/f0 в реальном времени через Pitchfinder YIN.
- Перевод частоты в MIDI и музыкальные ноты.
- Сглаживание и фиксация стабильной ноты.
- Пошаговый сценарий теста: нижняя нота, верхняя нота, результат.
- Отображение текущей ноты, частоты и отклонения в центах.
- Визуальная клавиатура C1-C8.
- График высоты тона за последние 10 секунд.
- Расчёт диапазона в полутонах и октавах.
- Примерная классификация: бас, баритон, тенор, контральто, меццо-сопрано, сопрано.
- Подбор похожих известных голосов из захардкоженного справочника.
- Экспорт результата в PNG и JSON.

## Требования

- Node.js 18+
- npm
- Современный браузер с поддержкой `getUserMedia`

Для доступа к микрофону сайт должен быть открыт по HTTPS. Исключение: `localhost`, где микрофон работает без HTTPS.

## Деплой

Проект полностью статический. Для публикации достаточно загрузить содержимое папки `dist` на любой статический хостинг:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- любой обычный хостинг со статическими файлами

Если приложение размещается не в корне домена, а в подпапке, например `https://example.com/voice/`, нужно настроить `base` в `vite.config.ts`.

## Как пользоваться

1. Откройте приложение.
2. Выберите микрофон в поле "Устройство захвата".
3. Нажмите "Начать тест".
4. Разрешите доступ к микрофону.
5. Спойте самую низкую комфортную ноту и удерживайте её до фиксации.
6. Подтвердите нижнюю ноту.
7. Спойте самую высокую комфортную ноту и удерживайте её до фиксации.
8. Подтвердите верхнюю ноту.
9. Посмотрите результат и при необходимости экспортируйте его в PNG или JSON.

## Важные ограничения

Классификация голоса приблизительная. Реальный тип голоса зависит не только от экстремальных нот, но и от тесситуры, тембра, переходных зон, техники и комфортного диапазона.

Подбор похожих известных голосов считается по пересечению диапазонов, близости центра диапазона, ширине диапазона и примерному типу голоса. Это развлекательная подсказка, а не профессиональная вокальная диагностика.
